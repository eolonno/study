Протокол тестирования

  • Имя тестируемого: %UserName%.
  • Класс: %UserGroup%.
  • Дата тестирования: %Date%.
  • Время начала: %TimeBegin%. Время завершения: %TimeEnd%.
  • Продолжительность: %TimeLength%.
  • Название теста: %TestTitle%.
  • Файл с тестом: %TestFile%.
  • Файл с тестом: %TestFileFull%.
  • UID теста: %TestUID%.
  • Всего заданий в тесте: %CountTask%. Выполнено заданий: %CountAskTask%.
  • Из них правильно: %CountCorrectTask% (%CountCorrectTaskPercent%% выполненных заданий.)
  • Из них ошибок: %CountErrorTask% (%CountErrorTaskPercent%% выполненных заданий).
  • Результативность: %Effectiveness%%.
  • Использовано подсказок: %CountUsePrompt%.
  • Набрано баллов: %Score% из %ScoreMax% возможных. Результат: %ResultScore%%.
  • Оценка: %Mark%.
  • %MaskOfResult%
  • %MaskOfTime%
  • %MaskUserAnswer%
  • %MaskOfPromptPenalty%

Таблица результатов

%TabResult% row[Result, Time]

Таблица результатов по группам заданий

%TabGroupResult% col[Effectivness, Right, Error, Miss, Time, Score]



Дата: %DateNow% %TimeNow%.
